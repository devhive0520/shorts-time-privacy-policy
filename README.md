# Shorts Time Privacy Policy

쇼츠타임(Shorts Time) 서비스의 개인정보 처리방침 페이지입니다.

## 파일 구조

- `index.html` - 언어 자동 감지 및 리다이렉트 페이지
- `privacy-policy-ko.html` - 한글 버전 개인정보 처리방침
- `privacy-policy-en.html` - 영어 버전 개인정보 처리방침

## 기능

- 브라우저 언어 설정에 따라 자동으로 적절한 언어 버전으로 리다이렉트
- 수동으로 언어를 선택할 수 있는 옵션 제공
- 반응형 디자인으로 모바일 및 데스크톱 환경 지원

## 사용 방법

1. `index.html`을 웹 서버에 배포하세요.
2. 사용자가 접속하면 브라우저 언어 설정에 따라 자동으로 적절한 언어 버전으로 이동합니다.
3. 직접 특정 언어 버전에 접근하려면:
   - 한글: `privacy-policy-ko.html`
   - 영어: `privacy-policy-en.html`
